Paragraf di HTML

Saat ini kita sedang berlatih dasar HTML. pada mata kuliah Pemrograman Web. Materi awal adalah membuat tampilan web sederhana menggunakan tag dasar HTML.

Tujuan mempelajari HTML adalah agar mampu membuat dan mengembangkan halaman web yang terstruktur dan menarik.

Format Tulisan

Contoh penggunaan teks tebal, teks miring, teks dicoret.

Baris Baru vs Paragraf

Tag <br> hanya membuat baris baru.
Contoh baris kedua menggunakan <br>.
